## 🔧 Adding Tools to Your Agent

Tools let agents perform specific actions. มาเพิ่ม tool สำหรับตรวจสอบ vehicle availability:

```python
# agent.py
from google.adk.agents.llm_agent import Agent

# Define a tool function - ต้อง type hints
def check_vehicle_availability(model: str, color: str) -> dict:
    """Check if a specific vehicle model and color is in stock.
    
    Args:
        model: Vehicle model name (e.g., 'Toyota Camry', 'Honda Civic')
        color: Vehicle color (e.g., 'white', 'black', 'silver')
    
    Returns:
        dict with availability info
    """
    # Simulated inventory database
    inventory = {
        'Toyota Camry': {'white': 3, 'black': 2, 'silver': 1},
        'Honda Civic': {'red': 2, 'blue': 1, 'white': 2},
        'Toyota Vios': {'silver': 5, 'black': 3}
    }
    
    if model in inventory and color in inventory[model]:
        count = inventory[model][color]
        return {
            'available': True,
            'model': model,
            'color': color,
            'in_stock': count,
            'message': f'We have {count} {color} {model}(s) available for viewing'
        }
    else:
        return {
            'available': False,
            'model': model,
            'color': color,
            'message': f'Sorry, {color} {model} is not currently in stock. We can order one for you.'
        }

def get_vehicle_price(model: str) -> dict:
    """Get the price of a specific vehicle model.
    
    Args:
        model: Vehicle model name
    
    Returns:
        dict with pricing information
    """
    prices = {
        'Toyota Camry Hybrid': {'price': 1299000, 'currency': 'THB'},
        'Honda Civic Type R': {'price': 2499000, 'currency': 'THB'},
        'Toyota Vios': {'price': 599000, 'currency': 'THB'}
    }
    
    if model in prices:
        price_info = prices[model]
        return {
            'model': model,
            'price': price_info['price'],
            'currency': price_info['currency'],
            'message': f'{model}: ฿{price_info["price"]:,}'
        }
    else:
        return {
            'available': False,
            'message': f'Model "{model}" not found in our catalog'
        }

# Create agent with tools
sales_agent = Agent(
    model='gemini-1.5-flash',
    name='AutoSalesAgent',
    description='Sales assistant for automotive dealership',
    instruction="""You are a professional automotive sales assistant.
You have access to tools to check vehicle availability and pricing.
Always use the tools to provide accurate information to customers.""",
    tools=[
        check_vehicle_availability,
        get_vehicle_price
    ]
)
```

## 📝 How It Works

### Agent Flow

```
User Input
    ↓
┌─────────────────────────┐
│  Gemini Model processes │
│  input with context     │
└─────────────────────────┘
    ↓
┌─────────────────────────┐
│  Decide if tool needed? │
└─────────────────────────┘
    ↓ (If yes, call tool)
┌─────────────────────────┐
│  Execute Tool Function  │
│  (e.g., check_vehicle)  │
└─────────────────────────┘
    ↓
┌─────────────────────────┐
│  Return tool result to  │
│  Gemini for response    │
└─────────────────────────┘
    ↓
User gets response
```

## 💡 Key Concepts

### Agent Component
- **model**: LLM ใช้ (เลือก gemini-1.5-flash)
- **name**: ชื่อ agent
- **description**: อธิบายหน้าที่
- **instruction**: System prompt ให้เอเจนต์
- **tools**: List ของ functions เอเจนต์ใช้ได้

### Tool Requirements
- ต้องมี Python type hints (`str`, `int`, `dict`, ฯลฯ)
- ต้องมี docstring อธิบายพารามิเตอร์
- ต้อง return ค่ำใช้ได้ (dict, str, int, etc)

---

## 🎯 Best Practices

1. **Clear Instructions**: ให้ instruction ชัดเจน directive ต่อ agent
2. **Type Hints Required**: Tools ต้องมี type hints ทั้งหมด
3. **Useful Docstrings**: Docstring บอกให้ agent รู้ว่า tool ทำอะไร
4. **Realistic Tools**: Tool ต้องตอบสนอง real use cases
5. **Error Handling**: Return meaningful messages เมื่อเกิด error

## 📊 Performance Monitoring

### Basic Metrics

```javascript
class MonitoredAgent extends BasicAgent {
  constructor() {
    super();
    this.metrics = {
      requests: 0,
      errors: 0,
      avgResponseTime: 0
    };
  }

  async respond(message) {
    const startTime = Date.now();
    this.metrics.requests++;

    try {
      const response = await super.respond(message);
      const duration = Date.now() - startTime;

      // Update average response time
      this.metrics.avgResponseTime =
        (this.metrics.avgResponseTime + duration) / 2;

      return response;
    } catch (error) {
      this.metrics.errors++;
      throw error;
    }
  }

  getMetrics() {
    return this.metrics;
  }
}
```
