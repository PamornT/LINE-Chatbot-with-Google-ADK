## 🎯 Objective
สร้าง AI agent แรกด้วย Google ADK (Python) - ใช้งาน framework สำหรับ agent development

---

## 🛠️ Prerequisites

ก่อนเริ่ม ตรวจสอบว่า:
- ✅ Codespace พร้อมใช้งาน
- ✅ Gemini API key กรณีที่ยังไม่มีให้ไปสร้างที่ ([Google AI Studio](https://aistudio.google.com/))
- ✅ Python 3.10+ ทำงานได้

```bash
python3 --version
pip --version
```

---

## 📦 Installation & Project Setup

### 1. Install Google ADK

```bash
pip install google-adk
```

### 2. Create Agent Project

Google ADK มี CLI tool เพื่อสร้าง project structure:

### สร้าง folder workshop
```bash
mkdir workshop
cd workshop
```

### create agent ผ่าน Google ADK CLI
```bash
adk create sales_agent
```

Project structure ภายใน folder workshop จะเป็นแบบนี้:
```
sales_agent/
├── agent.py           # Main agent code
├── __init__.py
└── .env              # Environment variables (API keys)
```

---

### Running Your First Agent

ทดสอบ agent ใน 2 วิธี:

#### วิธี 1: Command-line Interface

```bash
# รัน agent ผ่าน CLI
adk run sales_agent

# จากนั้น พิมพ์คำถามได้เลย
```

#### วิธี 2: Web Interface (Recommended)

```bash
# รัน web interface บน localhost:8000
adk web --port 8000

# เปิด browser ไปที่ http://localhost:8000
```

## 🤖 First Agent: Basic Structure

### Default Generated Agent (agent.py)

Google ADK สร้าง agent ดังนี้:

```python
# agent.py - generated by adk create
from google.adk.agents.llm_agent import Agent

# สร้าง root agent - element ที่จำเป็น
root_agent = Agent(
    model='gemini-2.5-flash',
    name='SalesAgent',
    description='A helpful sales assistant for automotive dealership',
    instruction='You are a helpful sales assistant for cars. Be professional and assist customers with their inquiries.'
)
```

## 🎯 Automotive Sales Agent Example

ลองกำหนดหน้าที่ให้ Agent ตอบคำถามเกี่ยวกับเรื่อง Automotive:

```python
# agent.py
from google.adk.agents.llm_agent import Agent

# สร้าง sales agent สำหรับ dealership
root_agent = Agent(
    model='gemini-2.5-flash',
    name='AutoSalesAgent',
    description='Sales assistant for automotive dealership in Thailand',
    instruction="""You are a professional automotive sales assistant for a Thai car dealership.
    
Your responsibilities:
- Help customers find the right vehicle
- Answer questions about car features, specifications, and pricing
- Suggest suitable models based on customer needs
- Provide information about financing options
- Be courteous and professional

Always respond in Thai when customer speaks Thai, English when appropriate.
Provide pricing in Thai Baht (฿).

Popular models we offer:
- Toyota Camry Hybrid: ฿1,299,000 (sedan, fuel-efficient)
- Honda Civic Type R: ฿2,499,000 (sports car, high performance)
- Toyota Vios: ฿599,000 (city car, economical)

Make suggestions based on customer's stated needs and budget.""",
)
```
